# 참고 1. 실험 데이터 및 구성

□ 생산, 투자 실시간 경기 진단 모형에 활용된 것과 동일한 데이터를 활용
◦ 전체 실험에 있어 동일한 파이프라인으로 데이터 전처리(InvestmentData, PRoductionData 클래스로 정의)
◦ 두 목표변수 모두 강한 트랜드를 보이며 Bimodal 분포가 나타남
◦ 전 산업 생산지수의 경우 변수들과 강한 상관성이 나타났으며, 두 모델에 포함된 시계열 데이터에서 최대 65%의 결측치 확인
◦ 전 산업 생산지수의 작은 분산과 강한 변수들 간 상관성으로 인해 모델이 학습할 수 있는 정보가 부족하여 동적 요인 모델 수렴 어려움
